String:

String is sequence data type in python.String is a collection of alphabets, words or other
characters.Python has a built-in string class named str.

To represent a string, you wrap it within quotes. There can be multiple ways of doing this:
Single quotes, for example:
'This is single string "double" quotes in your string.'
Double quotes. for example:
"This is double string 'single' quotes in your string."
Triple quotes, for example:
"""This is triple string""", '''Triple quotes using single quotes.'''
